=====================================
9. Pseudorandom number generator
=====================================

 > 10. Noise generator
This toolkit is an implementation of three fast and high quality pseudorandom number generators:
  • a Mersenne twister generator
  • a Complementary-Multiply-With-Carry generator,
  • a Generalised Feedback Shift Register.

CMWC and GFSR are faster than MT (see table below). CMWC has a much better period than MT (1039460 vs. 106001), while GFSR is somewhat behind the other two (102917). CMWC is the default algo since libtcod 1.5.0.

For python users :
Python already has great builtin random generators. But some parts of the Doryen library (noise, heightmap, ...) uses RNG as parameters. If you intend to use those functions, you must provide a RNG created with the library.
